# Zip 益智遊戲求解器挑戰

本專案致力於探索、開發並比較多種解決 "Zip" 益智遊戲的演算法。專案內包含了一整套求解器，從精確演算法到元啟發式演算法，以及一個程序化謎題生成框架。

目前，本專案正在轉向服務導向架構，目標是為使用者提供一個可以直接互動的解謎應用程式。

## 關於 "Zip" 益智遊戲

遊戲的目標是畫一條連續的路徑，該路徑必須走過網格中所有可走的儲存格，且每個儲存格只能走一次。

### 遊戲規則
*   路徑必須覆蓋所有可走的儲存格，且每個儲存格只能訪問一次。
*   路徑必須按照數字順序（1 → 2 → 3 ...）連接所有標有數字的「路徑點」。
*   路徑不能穿過由 `|` 或 `—` 標示的牆壁。

## 主要功能

### 多種求解演算法
本專案實現了多種求解器，可分為以下幾類：

#### 1. 精確求解器 (Exact Solvers)
這些演算法保證能找到最佳（且唯一）的解，但對於大型或複雜的謎題可能會非常耗時。
*   **深度優先搜尋 (DFS)**: 一種基於回溯的求解器，會窮盡所有可能的路徑。
*   **A* 搜尋**: 一種更高效的圖形遍歷演算法，使用啟發式函數（曼哈頓距離）來引導其搜尋方向。
*   **約束程式設計 (CP-SAT)**: 使用 Google 的 OR-Tools 將謎題建模為約束滿足問題，以找到有效的解。

#### 2. 元啟發式求解器 (Metaheuristic Solvers)
這些演算法使用機率和最佳化技術來快速找到高品質的解，特別適用於那些難以在合理時間內找到精確解的超大型或複雜謎題。
*   **蒙地卡羅搜尋 (Monte Carlo Search)**
*   **模擬退火 (Simulated Annealing)**
*   **遺傳演算法 (Genetic Algorithm)**
*   **禁忌搜尋 (Tabu Search)**
*   **粒子群最佳化 (Particle Swarm Optimization)**

### 程序化謎題生成
專案中包含一個強大且支援多進程的腳本 (`src/core/puzzle_generator.py`)，能夠生成大量帶有保證解的謎題資料集。

### 解法視覺化
本專案提供多種視覺化工具來展示解題路徑：
*   靜態的終端機網格列印。
*   在終端機中逐步播放的動畫。
*   將解題動畫匯出為高品質的 GIF 檔案。

## 目前狀態與下一步

專案的當前重點正從純演算法開發轉向建立一個面向使用者的服務。

*   **新方向**: 使用 **FastAPI** 作為後端邏輯，並搭配 **Gradio** 建立互動式使用者介面。這將允許使用者上傳自己的謎題並獲得解答。
*   **暫停中的開發**: 先前曾探索使用**強化學習 (RL)** 來解決謎題。這部分工作目前已**暫停**，以待後續進一步研究。詳細的開發進度與挑戰記錄在開發日誌中。

## 開始使用

### 環境需求
*   Python 3.11
*   [uv](https://github.com/astral-sh/uv) (用於套件管理)

### 安裝
複製本專案的儲存庫，並使用 `uv` 同步環境：
```bash
# 此指令將會從 pyproject.toml 和 uv.lock 安裝所有依賴套件
uv sync
```

### 運行測試
若要運行完整的測試套件並生成報告：
```powershell
.\run_tests.bat
```
測試結果和詳細日誌將儲存在 `src/core/tests/reports/` 目錄中。
